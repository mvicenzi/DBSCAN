# DBSCAN
Simple C++ implementation of DBSCAN for n-th dimensional points.

## How to build 
Standard `cmake` + `make` build. No major dependencies.

What you should typically want to do:
* Create a build subdirectory `mkdir build` and `cd` to it;
* Run `cmake -B . -S ../.`;
* Run `make`.
